# Mòdul 1: Introducció i fonaments

## Què és Claude AI?
Claude AI és un assistent d’intel·ligència artificial avançat desenvolupat per Anthropic, basat en models de llenguatge gegants (LLMs). Es caracteritza per la seva gran finestra de context, seguretat i capacitat de col·laboració complexa.

### Característiques clau:
- Finestra de context fins a 200K–1M tokens.
- Mode “Extended Thinking” per raonament de gran profunditat.
- Integració nativa d’skills, agents, MCP i eines externes.
- Opcions web, API i CLI/Claude Code.

## Plans i ecosistema
- **Pla Free:** ús limitat, només model Sonnet, sense extensió de context ni projects.
- **Pla Pro:** +5x límit, extended thinking i projects.
- **Pla Max:** +25x/+100x límit, accés preferent, memòria persistent, funcions experimentals.

## Primer contacte: passos inicials
1. **Registra’t a [claude.ai](https://claude.ai).**
2. Explora la interfície amb un project senzill.
3. Prova diferents models (Haiku, Sonnet, Opus).
4. Crea la teva primera consulta amb Extended Thinking.
5. Consulta “Settings” per veure la teva quota i preferències.

## Exemple pràctic: primera consulta

– Explica’m els avantatges de la finestra de context gran a Claude.

**Resposta típica**:  
La finestra de context gran permet mantenir context en converses llargues, processar documents extensos i millorar la coherència d’historial en projectes complexos.

## Q&A
- **Què diferencia Claude d’altres LLMs?**  
  – La finestra de context gegant i la integració nativa d’eines avançades.
- **Es pot utilitzar sense API Key?**  
  – Sí, via web i CLI si tens subscripció.
- **En què consisteix “Extended Thinking”?**  
  – És la capacitat de gastar molts més tokens per pensar i resoldre problemes difícils.
